# Release v2.7.0 (Retrospective)

## Scope (Які фічі було додано?)
- Перехід на **OLMUI Архітектуру** для CLI застосунків.
- Базове налаштування `ProjectBlueprint` (генератора проєкту) з інтерактивним вводом.
- Виправлення багів з `[object Object]` для порожніх об'єктів.
- Виправлення бага TypeScript для інтерфейсів (`BodyClass['UI']`).
- Помилка `FormInput.type = invalid 'array'` замінена на обробку типів `text` або `multiselect` у `ui/form.js`.
- Директорія CLI генераторів очищена (blueprint переїхав в `src/blueprint`).
- Адаптер `nan0cli.js` більше не зациклюється вічно після обробки одиничної команди (додано `break`).

## Acceptance Criteria
- [x] `nan0cli blueprint` працює автономно.
- [x] Форми коректно перетворюють `string[]` на `multiselect`.
- [x] Модель і конструктори інкапсулюють змінні та дефолтні значення через `resolveDefaults`.
- [x] `knip` сканує і перевіряє директорію `blueprint` у загальному просторі пакету.

*Примітка: Цей реліз оформлений пост-фактум (retrospective) після успішної стабілізації коду в межах робочої сесії.*
