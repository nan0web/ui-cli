# Release v2.5.0 — Event-Driven PlaygroundTest

## Місія

Прискорити виконання Sandbox Snapshot тестів (`npm run test:snapshot`). Замість In-Process підходу (Option 1), який порушує сумісність Node.js tty/readline для інтерактивних кастомних UI-компонентів, реалізовано **Варіант 2 (Event-Driven PlaygroundTest)**.

## Scope

1. **`src/test/PlaygroundTest.js`**:
   - Видалення жорсткого таймера `delayMs = 120`.
   - Додавання Event-Driven логіки `#feedSequence` з аналізом `stdoutBuffer` для виявлення маркерів CLI (наприклад, `? `, `> ` або нових рядків `\n`), що сигналізують про закінчення рендеру конкретного фрейму.
   - Миттєва передача наступного вводу в `stdin.write()`, зменшення циклу до ~20ms на фрейм.

2. **Загальна оптимізація**:
   - Замість >45-60 секунд, тести виконуються ~16 секунд зі 100% збереженням ізоляції (уникнення ESM витоків стану).
   - Оновлені `sandbox` снепшоти.
