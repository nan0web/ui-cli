## 🧭 Фаза 1: Філософія та Ідентичність

Цей пакет слугує **Канонічною Реалізацією** [стандартів nan•web UI](../ui/project.md).

### 1. Місія та Модель Взаємодії

**UI-CLI** — це не просто інструмент; це прояв принципу **"Взаємодія як Двигун"**:

- **Взаємодія як Діалог:** Кожен додаток — це безперервний цикл запитань і відповідей. UI-CLI формалізує цей діалог у терміналі.
- **Мінімалістична Істина:** Якщо бізнес-логіку неможливо виразити через CLI (використовуючи `Input`, `Select`, `Table`, `Tree`), логіка ще не повністю формалізована.
- **Фундамент для Чату:** Конверсаційні ШІ-інтерфейси (`ui-chat`) будуються безпосередньо через **використання `ui-cli` як інструмента ведення чату** (LLM оперує формальними примітивами `ui-cli` замість того, щоб просто генерувати вільний текст).

### 2. Межі Імплементації

- **Ізоляція Логіки:** Усі UI компоненти є виключно "безстановими" (stateless) рендерами `props`.
- **Агностичність до Середовища:** Основна логіка знаходиться в `InputAdapter`, який може бути замінений для інших термінальних середовищ або тест-раннерів.
- **Стандарт Golden Master:** Ідеальна візуальна консистентність між EN/UK локалями гарантується 34 автоматизованими знімками (snapshots).

## 🎨 Фаза 2-4: Візуальна та Технічна ДНК

### 🛠️ Архітектура: Дотримання "Конституції"

Ми суворо дотримуємося моделей, описаних у [`ui/project.md`](../ui/project.md):

| Стандарт             | CLI Реалізація                                                  |
| :------------------- | :-------------------------------------------------------------- |
| **SandboxVariant**   | Реєструється у `.cli-sandbox.json` зі збереженням `props`.      |
| **SandboxConfig**    | Реалізує `list` режим перегляду, оптимізований для терміналу.   |
| **Universal Blocks** | Перетворює стандартні AST-вузли в інтерактивні блоки `prompts`. |

### 🧩 Матриця Компонентів

Наш "Залізний Реєстр" включає 19 активних компонентів:

- **Views (7):** Alert, Badge, Toast, Table, Tabs, Breadcrumbs, Steps.
- **Prompts (12):** Input, Password, Toggle, Confirm, Select, Multiselect, Slider, DateTime, Tree, Spinner, ProgressBar, Sortable.
